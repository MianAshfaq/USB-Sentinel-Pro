# Security Policy

Please do not publish exploitable security defects in a public issue. Report
them privately through the repository owner's GitHub security advisory page.

Include the affected version, Windows version, reproduction steps, expected
behavior, actual behavior, and any relevant sanitized logs. Never include real
passwords, private files, malware samples, or personal information.

USB Sentinel Pro is defensive software. Reports or contributions that add
offensive capability, credential theft, persistence abuse, or unauthorized
access will not be accepted.
