# Contributing

Contributions are welcome under the MIT License.

1. Open an issue describing the bug or proposed defensive feature.
2. Keep privileged operations in the Windows Service and UI operations in the
   WPF project.
3. Preserve fail-closed behavior and never log passwords or private file data.
4. Add focused tests for protocol or security behavior.
5. Run `scripts\build.ps1` before submitting a pull request.

By contributing, you agree that your contribution may be distributed under
the project's MIT License.
